# Sample Tech Radar

Tech Radar example using [Jekyll](https://jekyllrb.com/) and [blips-chart](https://github.com/davideicardi/blips-chart).

## Requirements

- Ruby version 2.5.0

## Build

```
bundle exec jekyll build
```

## Development

```
bundle exec jekyll serve --livereload
```

Open `http://127.0.0.1:4000`.
